08 / PRESENTER GLOSSARY

What every number means

The page uses only trace fields, deterministic groupings, and optional prices you provide.

Agent event
A recorded triage, fix, or test-writing step where an AI worker may incur cost. Cost coverage is the share of these events carrying a cost sample.
Estimated USD
A dollar value calculated only for a token-only sample whose model appears in your price table. It is always labeled “estimated at your prices.”
Fingerprint
A stable ID for one failure. Fix events sharing it are treated as attempts on the same incident.
Harness
The surrounding tool that starts an AI worker and records its usage, such as Claude Agent SDK, Codex, or Grok.
Incident
One distinct bug. The dashboard uses the dedupe stage’s explicit incident count, or fingerprint count only when that explicit field is absent.
Model
The named AI system used by a cost sample. Prices match this exact string.
Needs human
An incident routed directly to a person plus an automated incident handed off after its allowed fix attempts fail.
Partial cost data
At least one agent event has no cost sample. Known dollars are shown, but the badge warns that they are not a complete run total.
Reported USD
A dollar amount present in the trace itself. The dashboard sums only samples that actually carry the usd field.
Retry
A fix attempt after the first for the same fingerprint. An un-fingerprinted attempt counts as a retry only when its explicit attempt number is above one.
Seam
A replaceable pipeline boundary. Spend uses triage, fix, test-writing, and other; time uses triage, fix, verification, and other.
Token
A unit of model input or output text. Input and output counts are kept separate for each harness and model.
Verified fix
A verify-stage event with the exact normalized outcome verified, pass, or passed. Current pipeline emitters use verified.
Wall duration
The non-negative difference between a run’s finishedAt and startedAt timestamps.
USD per verified fix
All known reported and user-priced cost in the selected run or loaded set, divided by its verified fix count. It is unavailable when either side is missing or zero.
